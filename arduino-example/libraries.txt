Required Libraries for Smart Tank ESP32:

1. WebSockets (by Markus Sattler) - Version 2.6.1 or later
   - Install from: https://github.com/Links2004/arduinoWebSockets
   - Or via Arduino IDE Library Manager

2. ArduinoJson (by Benoit Blanchon) - Version 7.4.2 or later
   - Install from: https://github.com/bblanchon/ArduinoJson
   - Or via Arduino IDE Library Manager

3. ESP32 Built-in Libraries (included with ESP32 Arduino Core):
   - WiFi
   - WebServer
   - Preferences
   - LittleFS
   - esp_now
   - FS

Installation Instructions:
1. Open Arduino IDE
2. Go to Tools -> Manage Libraries
3. Search for "WebSockets" and install
4. Search for "ArduinoJson" and install
5. Make sure ESP32 Arduino Core is installed (Tools -> Board -> Boards Manager -> search "ESP32")

Note: AsyncWebServer library is NOT required. The code uses the built-in WebServer library instead.
